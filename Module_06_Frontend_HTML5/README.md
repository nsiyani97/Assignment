# HTML5

## THEORY ASSIGNMENT

### 1. Difference between HTML and HTML5

- HTML is the earlier version of HyperText Markup Language used to structure web pages.
- HTML5 is the latest version that introduces new features for modern web applications.
- HTML5 supports multimedia elements like audio and video without using external plugins.
- HTML5 provides better support for mobile devices and responsive design.
- It also includes new semantic elements that improve accessibility and SEO.

### 2. What are the additional tags used in HTML5?

- HTML5 introduces new semantic and media-related tags that improve structure, accessibility, and functionality.
- These tags help browsers and developers better understand content meaning and support rich media.
- Common HTML5 semantic tags include `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.
- Multimedia tags like `<audio>` and `<video>` are added to embed media content easily.
- Graphics-related tags such as `<canvas>` and `<svg>` allow drawing and animations.
- Other useful tags include `<figure>`, `<figcaption>`, `<details>`, `<summary>`, `<progress>`, `<meter>`.
