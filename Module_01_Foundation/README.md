# Foundation

## THEORY EXERCISE

### 1. What is HTTP?

- HTTP (HyperText Transfer Protocol) is a communication protocol used between a client (browser) and a server.
- It defines how requests are sent from the browser and how responses are received from the server.
- HTTP is a stateless protocol, meaning each request is handled independently.
- It is mainly used to transfer web pages, images, videos, and data over the internet.

### 2. What is a Browser? How does it work?

- A browser is a software application used to access, retrieve, and display websites on the internet.
- Examples of browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari.
- When a user enters a URL, the browser sends an HTTP request to the web server.
- The server responds with files like HTML, CSS, and JavaScript, which the browser interprets and displays as a web page.

### 3. What is a Domain Name?

- A domain name is the easy-to-read address of a website, such as example.com.
- It helps users access websites without remembering numerical IP addresses.
- Domain names are connected to IP addresses using the Domain Name System (DNS).
- A domain name represents a website’s identity and is important for branding and accessibility.

### 4. What is Hosting?

- Web hosting is a service that provides space on a server to store website files and data.
- Hosting allows a website to be accessible on the internet 24/7.
- When a user visits a website, hosting servers deliver the website files to the browser.
- Hosting services also provide features like security, backups, and server maintenance.
